Here are some steps to push a change in git

  1. Make some changes in any way in your repo
  2. Stage those changes by using git add .
  3. Commit those changes using git commit -m "Your commit message"
  4. After commiting changes push it to something like Github by using git push origin main

After pushing those changes you check status of your git repo using git status and also check history of commits using git log
